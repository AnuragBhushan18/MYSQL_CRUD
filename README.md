\# MYSQL CRUD Project



\## Overview



This project demonstrates CRUD (Create, Read, Update, Delete) operations using Java JDBC and MySQL.



\## Features



✅ Create new records  

✅ Read all records  

✅ Read a specific record  

✅ Update records  

✅ Delete records  

✅ Alter table structure  

✅ Truncate table data  



\## Technologies



\- Java

\- JDBC

\- MySQL

\- Eclipse IDE

\- Git

\- GitHub



\## Database Operations



| Operation | SQL Command |

|------------|------------|

| Create | INSERT |

| Read | SELECT |

| Update | UPDATE |

| Delete | DELETE |

| Alter Table | ALTER TABLE |

| Truncate Table | TRUNCATE TABLE |
